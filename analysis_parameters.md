# Analysis Parameters for Digital Twin

This document outlines the key parameters considered for analysis within the CNC machining digital twin.

## Machine-Level Parameters
- Machine ID
- Machine utilization
- Operation frequency

## Operation-Level Parameters
- Operation ID (Turning, Facing, Drilling, etc.)
- Cycle time per operation
- Operation sequence

## Process Parameters
- Spindle speed (RPM)
- Feed rate
- Tool condition / wear indicator

## Analytical Focus
- Machine-wise performance comparison
- Operation-wise efficiency analysis
- Identification of abnormal or deviating patterns

These parameters provide the analytical backbone of the digital twin and can be expanded with real-time and sensor-based inputs in future stages.
